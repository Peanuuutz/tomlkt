operator fun TomlTable.get(vararg keys: Any?): TomlElement?(source)

Gets the element along the path constructed by keys.

Given the following TOML file:

i = 0

[data]
list = [
    "any",
    { type: "integer", value: 0 }
]
Content copied to clipboard

With keys = ["data", "list", 1, "value"], the above table emits 0 (in TomlLiteral).

Parameters

keys

the path which leads to the value. Each one item is a single segment. If a TomlArray is met, any direct child segment must be Int or String (will be parsed into integer).
