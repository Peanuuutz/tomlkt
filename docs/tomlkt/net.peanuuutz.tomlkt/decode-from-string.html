decodeFromString

inline fun <T> Toml.decodeFromString(string: String, vararg keys: Any?): T(source)

Parses string into a TomlTable and deserializes the corresponding element fetched with keys into a value of type T using serializer retrieved from reified type parameter.

Parameters

string

MUST be a TOML file, as this method delegates parsing to Toml.parseToTomlTable.

keys

the path which leads to the value. Each one item is a single segment. If a TomlArray is met, any direct child segment must be Int or String (will be parsed into integer).

See also

get

Throws

TomlDecodingException

if string cannot be parsed into TomlTable or the element cannot be deserialized.

NonPrimitiveKeyException

if provided non-primitive keys.
