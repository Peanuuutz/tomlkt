fun <T> Toml.decodeFromNativeReader(deserializer: DeserializationStrategy<T>, nativeReader: Reader): T

Deserializes the content of the nativeReader into a value of type T using deserializer.

Note that when finished, nativeReader is automatically closed.

fun <T> Toml.decodeFromNativeReader(deserializer: DeserializationStrategy<T>, nativeReader: Reader, vararg keys: Any?): T

Parses the content of the nativeReader into a TomlTable and deserializes the corresponding element fetched with keys into a value of type T using deserializer.

Note that when finished, nativeReader is automatically closed.

Parameters

nativeReader

MUST contain a TOML file, as this method delegates parsing to Toml.parseToTomlTable.

keys

the path which leads to the value. Each one item is a single segment. If a TomlArray is met, any direct child segment must be Int or String (will be parsed into integer).
