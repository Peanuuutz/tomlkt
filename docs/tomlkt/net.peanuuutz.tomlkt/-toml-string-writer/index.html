TomlStringWriter

class TomlStringWriter : TomlWriter(source)

A TomlWriter that writes TOML as string.

Use with Toml.encodeToWriter to encode objects. This is useful for concatenating two TOML strings. However, programmer should prefer Toml.encodeToString if only one object is being encoded.

When finished, simply call toString to get the result.
