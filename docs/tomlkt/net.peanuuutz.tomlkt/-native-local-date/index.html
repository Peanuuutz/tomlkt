NativeLocalDate

expect class NativeLocalDate(source)

A date without a time-zone in the ISO-8601 calendar system.

For JVM, this is an alias of java.time.LocalDate. For other platforms, this is kotlinx.datetime.LocalDate.

This type only connects to platform-specific type, so is not serializable. See TomlLocalDate for the serializable one.
