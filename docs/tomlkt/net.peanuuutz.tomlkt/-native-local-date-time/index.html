expect class NativeLocalDateTime(source)

A date-time without a time-zone in the ISO-8601 calendar system.

For JVM, this is an alias of java.time.LocalDateTime. For other platforms, this is kotlinx.datetime.LocalDateTime.

This type only connects to platform-specific type, so is not serializable. See TomlLocalDateTime for the serializable one.
